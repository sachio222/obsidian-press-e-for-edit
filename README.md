# Press E for Edit

A minimal Obsidian plugin that adds Vim-style mode switching with just two keys.

## Usage

- **`e`** — Switch from reading mode to edit mode
- **`Escape`** — Switch from edit mode back to reading mode

That's it. No config, no settings, no keybinding conflicts. Works alongside Obsidian's default "Reading view" setting — files open in reading mode, press `e` when you want to edit.

## Why?

Obsidian doesn't have a quick keyboard shortcut to toggle between reading and editing. You either click the view toggle or set up a hotkey manually. This plugin makes it automatic — like Vim's `i` to insert, but simpler.

## Installation

Copy the plugin folder to `.obsidian/plugins/press-e-for-edit/` and enable it in Settings > Community Plugins.
